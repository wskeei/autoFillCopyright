# AutoFill Copyright Form (软著申请自动填写助手)

这是一个 Chrome 浏览器扩展插件，旨在帮助用户在申请软件著作权（软著）时，自动填写繁琐的表单信息，提高申请效率。

## 功能特性

该插件主要提供以下自动填写功能：

1.  **硬件/软件环境自动填充**：
    *   自动识别页面上的多行文本框（Textarea），并填充预设的硬件环境、软件环境、操作系统版本等信息。
    *   预设内容包括：CPU、内存、硬盘、带宽、Windows 10 操作系统、Node.js/Vue3 开发环境、Nginx/MySQL 版本等。
2.  **开发语言自动勾选**：
    *   自动查找并勾选 "JavaScript", "HTML", "SQL" 等复选框。
3.  **代码行数/数值随机生成**：
    *   在指定的输入框中生成 7000 到 10000 之间的随机数（通常用于填写代码行数）。
4.  **软件简介填充**：
    *   自动填充一段标准的软件系统特点描述（如：简单性、面向对象、健壮性等）。
5.  **一键操作**：
    *   通过浏览器插件弹窗中的“自动填写”按钮，一键完成当前页面的表单填充。

## 安装指南

1.  下载或 Clone 本仓库代码到本地。
2.  打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 进入扩展程序管理页面。
3.  开启右上角的 **"开发者模式" (Developer mode)**。
4.  点击左上角的 **"加载已解压的扩展程序" (Load unpacked)**。
5.  选择本项目的根目录（包含 `manifest.json` 的文件夹）。
6.  安装成功后，浏览器右上角将显示插件图标。

## 使用方法

1.  打开软著申请网站的相关填报页面。
2.  点击浏览器右上角的 **AutoFill Copyright Form** 插件图标。
3.  在弹出的窗口中，点击 **"自动填写"** 按钮。
4.  插件将自动查找页面中的对应表单元素并填入预设数据。

## 注意事项

*   **数据核对**：插件填充的数据为通用模板数据（如硬件配置、开发环境），**提交前请务必根据您的实际软件情况进行修改和核对**，确保申请信息的真实性和准确性。
*   **页面兼容性**：本插件是基于特定的页面结构（Class 名称如 `.fillin_info`, `.hd-text-area` 等）开发的，如果官网页面结构发生更新，插件可能会失效。
*   **隐私安全**：本插件仅在本地运行，不会上传您的任何填写信息。
